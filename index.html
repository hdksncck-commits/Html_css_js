دورات البرمجة

ابدأ رحلتك في البرمجة مع دوراتنا المبتدئة واصعد حتى الاحتراف

HTML

لغة بناء هيكل صفحات الويب

HTML هي اللغة الأساسية لبناء صفحات الويب. سوف تتعلم كيفية إنشاء هيكل الصفحة وإضافة النصوص والصور والروابط والنماذج.

CSS

لغة تنسيق وتصميم صفحات الويب

CSS هي اللغة المستخدمة لتنسيق صفحات HTML وتصميمها. ستتعلم كيفية جعل صفحات الويب جميلة وجذابة وسهلة الاستخدام.

JavaScript

لغة برمجة لإنشاء صفحات ويب تفاعلية

JavaScript تضيف التفاعل لصفحات الويب. ستتعلم كيفية إنشاء صفحات ويب تفاعلية وتطبيقات ويب كاملة.
